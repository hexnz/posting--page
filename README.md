# DevBlog

Projeto front-end para simular a criação de um post consumindo uma API aberta. A interface tem estilo de blog para desenvolvimento com uma vibe inspirada em terminal e matrix.

## Tecnologias

- HTML
- CSS
- JavaScript
- Fetch API

## Funcionalidades

- Campo para digitar o título do post.
- Campo para escrever o conteúdo.
- Envio dos dados com `fetch` usando o método `POST`.
- Renderização do retorno da API na própria página.
- Mensagem de status durante o envio.

## API utilizada

```txt
https://jsonplaceholder.typicode.com/posts
```

## Como executar

Faça o pull do projeto e Abra o arquivo `index.html` no navegador ou use uma extensão como Live Server no VS Code.
